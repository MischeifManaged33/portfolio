My Responsibilities

TerRand is a Unity plugin that transforms a simple plane into procedurally generated terrain using Perlin noise. It features automatic recoloring based on height variation and dynamically adjusts the level of detail (LOD) depending on the camera’s distance, optimizing performance while maintaining visual quality. Designed for flexibility and efficiency, TerRand enables seamless terrain generation for games and simulations, providing an easy-to-use solution for developers looking to create varied landscapes with minimal manual effort.
